|travis-img| tk-facebook ====== *simple facebook parsing*

Install
-------

::

    mkvirtualenv tk-facebook
    git clone https://github.com/newslynx/tk-facebook.git
    cd tk-facebook
    pip install -r requirements.txt
    pip install .

Test
----

Requires ``nose``

::

    nosetests
